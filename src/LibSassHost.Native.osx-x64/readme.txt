   ===========
   DESCRIPTION
   ===========
   This package complements the LibSassHost package and contains the native
   implementation of LibSass version 3.6.4 for OS X (x64).

   This package is only compatible with .NET Core.

   =============
   RELEASE NOTES
   =============
   Added support of the LibSass version 3.6.4.
